写在前面：这套方法是我帮在体制内工作的姐姐做了一份长征 90 周年党课汇报后总结出来的。她不会写代码，单位电脑装的是 WPS 和 Edge，最后她用我教她的提示词，半小时独立做出了第二份。

如果你也是这样的人，这篇就是写给你看的。

## 一、先看场合，别盲跟

这种 PPT 长什么样？就是你在 B 站、公众号上偶尔会看到的那种"网页版 PPT"——能放动画、能交互、配色比 PowerPoint 模板高级一截，全屏投影一打开就有"哎这家伙是不是请人做的"的效果。

但请先认清边界。**它适合：**

- 自己上去讲的会议、党课、培训、述职、节庆主题展示
- 用电脑或会议室投影仪，按方向键翻页
- 想让效果"亮一下"的场合

**它不适合：**

- 发给领导用 WPS / PowerPoint 改
- 打印出来归档
- 上传到内部 OA 系统作为成果材料

如果你的 PPT 最终命运是"发给领导改"，老老实实用 WPS 套模板，AI 只用来润色文字。这篇就别看了。

## 二、三步上手最简版

### 第 1 步：打开 DeepSeek

浏览器进 [chat.deepseek.com](https://chat.deepseek.com)，注册登录。免费、不用翻墙、不限速。

如果单位电脑装不了客户端、上得了网就行。微信扫码登录最快。

### 第 2 步：把下面这段提示词改两个字发给它

```
你是一位资深 PPT 设计师。请用一个完整的 HTML 文件帮我做一份汇报演示，主题是：{在这里填你的主题}。

要求：
- 单文件 HTML，所有 CSS 和 JS 都内联，不依赖任何外部资源
- 8 张幻灯片，每张占满全屏，按方向键翻页
- 风格庄重大气，配色用党旗红 + 烫金 + 米白，字体用宋体
- 包含：封面 / 背景介绍 / 关键数据可视化 / 时间轴 / 引用名言 / 收尾
- 每页要有动画进场效果
- 在右下角加翻页按钮和当前页码
- 关键数字用大字号突出
```

把 `{在这里填你的主题}` 换成你要做的题，比如"纪念长征胜利 90 周年党课汇报"。

### 第 3 步：把 AI 给的代码变成可以双击打开的文件

这一步最容易卡住，看仔细：

**Windows 用户：**

1. 桌面右键 → 新建 → 文本文档
2. 双击打开记事本
3. 把 AI 给的整段代码（从 `<!DOCTYPE html>` 到 `</html>`）全部复制粘贴进去
4. 文件 → 另存为
5. 文件名：`汇报.html`（注意：是 `.html` 不是 `.txt`）
6. 保存类型选"所有文件"，编码选 **UTF-8**
7. 双击这个文件，浏览器会自动打开

**Mac 用户：**

1. 打开"文本编辑"
2. 格式 → 转换为纯文本（`Shift + Cmd + T`）
3. 粘贴代码
4. 存储为 `汇报.html`
5. 弹窗提示后缀时选"使用 .html"

打开之后看到的就是你的 PPT，按方向键翻页。

## 三、5 套抄就能用的提示词

把上面"第 2 步"的主题部分，换成下面任意一个块。其他不要动。

### 党课 / 主题教育

```
{主题，例如：纪念红军长征胜利 90 周年}

要求同上，另外加：
- 第二页放一段习近平总书记的相关讲话原文（请你只写"[此处补充讲话原文]"占位，不要凭空生成）
- 倒数第二页归纳"对我们的启示"4 点
- 末页用一句振奋的诗词或口号收尾
```

### 工作总结 / 述职报告

```
{XX 单位 / XX 同志 2026 年第一季度工作总结}

要求同上，另外加：
- 第二页"主要指标"放 4 个关键数据，用大字号
- 中段放时间轴展示季度内的重要节点
- 倒数第二页"存在不足"3 条
- 末页"下阶段重点"3 条
```

### 学习贯彻类

```
{XX 精神 / XX 全会精神} 学习汇报

要求同上，另外加：
- 第二页"核心要义"分 3 块
- 中段"重点内容"用编号列表
- 倒数第二页"如何落实到岗位"
- 末页"我的体会"
```

### 节庆纪念

```
纪念{节日 / 历史事件}{XX} 周年

要求同上，另外加：
- 封面用最具代表性的视觉元素
- 第二页放一组震撼的数字
- 中段时间轴展示历史进程
- 末页用诗词或名言收尾
```

### 培训分享

```
{某主题}专题培训

要求同上，另外加：
- 第二页"为什么讲这个"
- 中段每张幻灯片一个核心概念，配示意图（用 SVG 画）
- 末页"3 条记住的话"
```

## 四、不满意？这样让 AI 局部改

新手最常犯的错误：每次不满意都让 AI 重写整个，然后越改越烂。

**正确姿势是局部改。** 把这些指令直接发给它：

- "把第 3 页标题改成 XXX，其他不动"
- "整体配色换成藏青 + 银白，更适合科技汇报"
- "在第 5 页之后插入一页，讲 XXX"
- "把第 6 页的字体放大 20%，标题加粗"
- "封面太花了，简化一下，只保留主题和年份"
- "第 4 页的时间轴改成竖向"
- "末页加上一句古诗词，要 14 字以内"

每次只改一处，AI 改完会重新发整段代码，你重新保存覆盖原文件即可。

## 五、投屏演示

会议室那台老电脑通常装的是 Edge 或 Chrome，都行。

1. 把 `.html` 文件拷到 U 盘，插会议室电脑
2. 双击打开
3. 按 **F11** 全屏
4. 方向键翻页，**Esc** 退出

提前 10 分钟到会议室连一遍投影仪，确认能正常播放。**别在领导面前才发现打不开。**

## 六、5 条红线（这部分救命）

这 5 条比上面的技巧都重要。AI 只是工具，最后挨批的是你。

**1. 政治站位 — 凡引用必查原文**

AI 会幻觉。它可能给你编一段听起来很像、但根本不存在的"总书记讲话"。任何引用都必须去人民日报、新华网、求是网核对原文，一字不差。

**2. 数据 — 凡数字必复核**

AI 编数字是常事。它给你的"6.5 万里"是真的，但它给你的"伤亡 8.6 万"可能是它编的。任何数字都要去权威来源核（统计公报、官方白皮书、人民网）。

**3. 涉敏材料 — 别往 AI 里发**

内部文件、未公开数据、单位红头文件、领导讲话稿，**一律不要发给任何 AI 工具**，包括 DeepSeek、豆包、Kimi、ChatGPT。这条没有例外。

只用 AI 做"通用框架 + 公开信息"，敏感内容你自己手动填到 HTML 里。

**4. 单位 / 领导姓名 — 让 AI 写就 100% 错**

AI 不知道你单位的全称、领导的具体职务、内部称呼习惯。这些**必须留占位**让你自己填。提示词里直接写"姓名职务请用 [XXX] 占位"。

**5. 政治术语 — 容易混淆的词列表**

AI 经常把这些写错或写串：

- "二十大"和"十九大"会混
- "中国共产党"漏写"中国"
- "习近平新时代中国特色社会主义思想"少字
- "第二个百年奋斗目标"和"第一个"搞反
- 涉及港澳台、民族、宗教的表述

凡是这一类，自己核一遍，不要信 AI。

## 七、进阶：要做出更高水准

把这段当作"长款"提示词用：

```
你是一位顶级 PPT 设计师，需要做出能在体制内汇报场合"惊艳全场"的演示。

主题：{你的主题}

设计要求：
- 单 HTML 文件，全内联
- 8 张幻灯片，左右键翻页，F 全屏
- 配色采用"党旗红 #a8201a + 烫金 #c9a961 + 宣纸米 #f5f0e6"，字体用 Noto Serif SC 衬线宋体配马善政手写体
- 每张幻灯片要有：纸纹背景纹理、章节序号水印、装饰性的篆刻印章元素
- 数字要做递增动画、关键节点要做时间轴、地理信息可以用 SVG 画路线图
- 引用区用左竖红线 + 米色底块
- 加一个顶部进度条和右侧圆点缩略导航

内容结构：
1. 封面（主题 + 年份 + 副标题）
2. 主题背景与历史意义
3. 关键数字 6 个（用卡片网格）
4. 历史脉络（时间轴形式）
5. 核心精神 / 16 字方针（4 宫格卡片）
6. 关键人物或事件回顾（带引文）
7. 路线图 / 流程图（用 SVG）
8. 收尾（诗词 + 行动号召）

请按这个规范输出完整代码。
```

这一段提示词背后藏的设计语言，是我自己琢磨了好几遍才稳定下来的。直接抄，比你自己描述快 10 倍。

## 八、最后说一句

AI 是杠杆，不是替代。

它能在 30 分钟内帮你把"码字 + 排版"的体力活做完，但**站位准不准、数据真不真、措辞分寸合不合适，永远是你自己的判断力**。

做完之后，自己从头到尾过一遍，过两遍，再交出去。

这是体制内的基本功，AI 替不了。
